# M18.5 — Vault Index System

**Status:** ✅ done

## Goal

Consolidate all vault indexes (search, backlinks, tags) into a single data structure built during one `ScanVault` pass. This prevents conflicting signature changes and improves performance.

## Motivation

Multiple milestones (M19 backlinks, M20 tags) need to modify `ScanVault` to add new return values. Without consolidation, we'd have:
- M19: adds `map[string][]string` for backlinks
- M20: adds `map[string][]string` for tags
- Conflicting changes to the same function
- Multiple passes over the vault

A unified index system solves both problems.

## Implementation Plan

### 1. Define VaultIndexes struct

New type in `vault.go`:

```go
type VaultIndexes struct {
    // Search index: path -> content (without frontmatter)
    Search map[string]string
    
    // Backlink index: normalized target -> source paths
    Backlinks map[string][]string
    
    // Tag index: tag name -> file paths
    Tags map[string][]string
}
```

### 2. Update ScanVault signature

Change from:
```go
func ScanVault(root string, skipDirs []string) (*VaultEntry, map[string]string, []string, error)
```

To:
```go
func ScanVault(root string, skipDirs []string) (*VaultEntry, *VaultIndexes, []string, error)
```

### 3. Build all indexes in single pass

During `WalkDir`, for each markdown file:
1. Read file content
2. Strip frontmatter, add to Search index
3. Parse frontmatter, extract tags, add to Tags index
4. Extract wiki-link targets, add to Backlinks index

```go
func ScanVault(root string, skipDirs []string) (*VaultEntry, *VaultIndexes, []string, error) {
    // ... existing walk logic ...
    
    indexes := &VaultIndexes{
        Search:    make(map[string]string),
        Backlinks: make(map[string][]string),
        Tags:      make(map[string][]string),
    }
    
    // For each markdown file:
    data, err := os.ReadFile(path)
    if err != nil { continue }
    
    content := string(data)
    
    // Search index
    indexes.Search[relPath] = stripFrontmatter(content)
    
    // Tag index
    tags := extractTagsFromFrontmatter(content)
    for _, tag := range tags {
        indexes.Tags[tag] = append(indexes.Tags[tag], relPath)
    }
    
    // Backlink index
    targets := extractWikiLinkTargets(content)
    for _, target := range targets {
        normalized := normalizeWikiLinkTarget(target)
        indexes.Backlinks[normalized] = append(indexes.Backlinks[normalized], relPath)
    }
    
    return tree, indexes, scanErrors, nil
}
```

### 4. Implement index builders

#### Tag extraction

```go
func extractTagsFromFrontmatter(content string) []string {
    fm, _ := parseFrontmatter(content)
    var tags []string
    for _, tag := range fm.Tags {
        // Normalize: strip #, lowercase
        tag = strings.TrimPrefix(tag, "#")
        tag = strings.ToLower(tag)
        if tag != "" {
            tags = append(tags, tag)
        }
    }
    return tags
}
```

#### Wiki-link target extraction

```go
var wikiLinkTargetRe = regexp.MustCompile(`\[\[([^\]|#]+)`)

func extractWikiLinkTargets(content string) []string {
    matches := wikiLinkTargetRe.FindAllStringSubmatch(content, -1)
    seen := make(map[string]bool)
    var targets []string
    for _, match := range matches {
        target := match[1]
        if !seen[target] {
            seen[target] = true
            targets = append(targets, target)
        }
    }
    return targets
}

func normalizeWikiLinkTarget(target string) string {
    // Lowercase, append .md if not present
    target = strings.ToLower(target)
    if !strings.HasSuffix(target, ".md") {
        target += ".md"
    }
    return target
}
```

## Edge Cases

- **Self-referencing links**: Note links to itself → appears in backlinks (legitimate)
- **Duplicate tags**: Same tag multiple times → deduplicate in extractTagsFromFrontmatter
- **Tag normalization**: `#Tag`, `tag`, `TAG` → all become `tag`
- **Wiki-link normalization**: `[[Note]]`, `[[note.md]]` → both become `note.md`
- **Empty frontmatter**: No tags → skip tag indexing for that file
- **No wiki-links**: No backlinks → skip backlink indexing for that file
