# M34 — Horizontal Scroll for Viewer

**Status:** ⏳ deferred

## Goal (deferred)

Viewer supports horizontal scrolling for content that exceeds viewport width (long code lines, wide frontmatter values, URLs). When scrolling horizontally, soft-wrap is temporarily disabled.

## Context

Deferred in favor of **M35** (Resizable Split). A resizable tree/viewer split solves the same practical problem — giving the viewer more room for wide content — with much less complexity. M34 remains in the plan for future consideration if users explicitly request horizontal scrolling after M35 is implemented.
